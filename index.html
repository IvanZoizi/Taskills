Проекты

Проект 1: SchoolBot

Бот взаимодействует с платформой Дневник.ру, предоставляя ученикам и родителям актуальную информацию о домашних заданиях, оценках, расписании и уведомлениях. Использует API для автоматизации запросов и удобного интерфейса взаимодействия через Telegram.

Проект 2: Bot_box_buy

Это универсальный бот для логистики, упрощающий процесс закупок и доставки товаров из Китая под ключ. Бот помогает заказчикам оформить заказ, рассчитать стоимость доставки, организовать отслеживание грузов и обеспечить полное сопровождение на всех этапах логистики. Интеграция с системами отслеживания грузов позволяет клиентам видеть статус доставки в реальном времени. Поддерживает удобный интерфейс через Telegram для мгновенной связи с менеджерами и автоматизированных расчётов.

Проект 3: ProfiTime

Веб-приложение для планирования мероприятий, позволяющее организаторам составлять расписание, управлять списком участников и отправлять автоматические напоминания. Интеграция с Google Calendar для удобного синхронизации.

Проект 4: AI Text Summarizer

Инструмент на основе искусственного интеллекта, который анализирует длинные тексты и генерирует краткие резюме. Поддержка нескольких языков и тематических настроек для адаптации под нужды пользователя (научные статьи, новости, бизнес-отчёты).

Проект 5: GameStats Tracker

Приложение для отслеживания статистики игроков в популярных онлайн-играх. Поддержка анализа прогресса, сравнительных таблиц и интеграция с игровыми API (Steam, Riot Games, Ubisoft). Идеально подходит для киберспортсменов и стримеров.
